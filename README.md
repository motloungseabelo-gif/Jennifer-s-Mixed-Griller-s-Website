# Jennifer-s-Mixed-Griller-s-Website
🍽️ Jennifer’s Mix Grillers Website

A modern, mobile-optimized restaurant website built to showcase Jennifer’s Mix Grillers, featuring a smooth UI, responsive design, and enhanced JavaScript interactivity.

🌟 Overview

This project is a fully responsive multi-page website designed for a catering and grill business.
It focuses on performance, user experience, and modern web design principles.

💡 “Where every plate feels like a celebration.”

🚀 Features
🎨 UI / Design
Orange sandstorm-inspired theme with warm sunlight tones ☀️
Clean, modern layout with glassmorphism elements
Smooth animations and transitions
Consistent typography and spacing
📱 Mobile Optimization
Fully responsive across all devices
Optimized navigation for mobile users
Touch-friendly buttons and layouts
Flexible grid system for content scaling
⚡ JavaScript Enhancements
Sticky navigation bar on scroll
Scroll reveal animations
Animated counters
Hero section parallax effect
Mobile menu toggle system
Form auto-save using localStorage
Performance-aware animations (respects reduced motion)
